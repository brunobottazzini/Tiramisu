# Tiramisù — UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Applicare 5 fix visivi alla `GameActivity` (tallone card-shape, theme colors casino, icona pausa, fondazioni più grandi con cornice slot, pile più strette).

**Architecture:** Modifiche solo a risorse XML (`game.xml`), un nuovo vector drawable (`ic_pause.xml`) e una nuova stringa. Nessuna modifica al codice Kotlin: `GameActivity.renderPile` calcola già le dimensioni delle carte da `container.width - padding`, quindi il restringimento delle pile si applica automaticamente.

**Tech Stack:** Android XML (layout + drawable + values), AAPT2 per validazione build. Nessuna dipendenza nuova.

**Spec di riferimento:** `docs/superpowers/specs/2026-05-21-ui-polish-design.md`.
